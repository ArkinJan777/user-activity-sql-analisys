# user-activity-sql-analysis

\## User Activity Analysis (SQL Project)



\### Project Overview

This project analyzes user activity data using SQL.

The goal is to understand user behavior, engagement, and activity patterns across different countries.



\### Key Questions

\- How many users are registered in each country?

\- Which countries have the highest user activity?

\- How active are users (sessions and events)?

\- What types of actions do users perform (login, like, comment, share)?



\### Dataset Description

The project is based on three main tables:



\- users — user information (country, signup date)

\- sessions — user sessions (date and duration)

\- events — user actions (login, like, comment, etc.)



\### SQL Skills Used

\- Data creation (CREATE TABLE)

\- Data insertion (INSERT)

\- Aggregations (COUNT, AVG, SUM)

\- Grouping (GROUP BY)

\- Filtering (WHERE)

\- Joins (JOIN)

\- Sorting (ORDER BY)



\### Key Insights

\- Ireland has the highest number of users in the dataset

\- User activity varies between countries

\- Some users are highly active with multiple sessions and events

\- Login and interaction events are the most common actions



\### Conclusion

This project demonstrates how SQL can be used to analyze user behavior and extract meaningful insights from raw data.
